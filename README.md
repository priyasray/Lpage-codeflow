## Getting Started

This project was bootstrapped with a standard **React** development setup.

* **Live Demo:** [https://priyasray.github.io/Lpage-codeflow/](https://priyasray.github.io/Lpage-codeflow/)

To run the project locally:

1. Clone the repository:
    ```bash
    git clone https://github.com/priyasray/Lpage-codeflow.git
    ```
2. Navigate to the project folder:
    ```bash
    cd Lpage-codeflow
    ```
3. Install dependencies:
    ```bash
    npm install
    ```
4. Start the development server:
    ```bash
    npm run dev  # Or 'npm start' depending on your setup
    ```
